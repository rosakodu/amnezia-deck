# Amnezia Deck

**Удобное использование Amnezia VPN на Steam Deck**

Инструкция по настройке и использованию Amnezia VPN на SteamOS — как в **режиме рабочего стола**, так и через **плагин в игровом режиме**.

## 🖥️ Режим рабочего стола

1. Скачайте и установите последнюю версию Amnezia VPN для Linux:  
   [AmneziaVPN_linux_x64.tar](https://github.com/amnezia-vpn/amnezia-client/releases)
2. Добавьте ваш конфиг в программу.

### Важные замечания

- **Обязательно** отключайте Amnezia перед переходом в игровой режим.
- После каждого обновления SteamOS Amnezia удаляется (конфиги сохраняются). Просто переустановите программу.
- Если забыли отключить Amnezia и Deck не включается — выключите роутер (или отключите Deck от роутера через мобильное приложение).

---

## 🎮 Игровой режим (Decky Loader)

1. Установите [Decky Loader](https://github.com/steamdeckhomebrew/decky-loader)
2. Скачайте плагин: [vpn-deck]([https://github.com/rosakodu/amnezia-deck/releases](https://github.com/MrWaip/vpn-deck/releases))
3. В настройках Decky Loader включите **режим разработчика**
4. Установите `.zip` архив плагина
5. Добавьте конфиг

---

## 🚀 Использование плагина

- **Импорт конфига** — кнопка «Импортировать конфиг», выберите файл `.conf`  
  Имя конфига — до 12 символов (буквы, цифры, `_`, `=`, `+`, `.`, `-`)
- **Включение / выключение** — переключатель напротив имени конфига
- **Удаление конфига** — кнопка «Удалить конфиг» (интерфейс будет остановлен)
- **Ошибки** — раздел «Ошибки»: просмотр истории и очистка
